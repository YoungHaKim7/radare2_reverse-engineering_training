# Windows Performance Analyzer 단계별 가이드
- 이 섹션에서는 WPA(Windows Performance Analyzer) UI(사용자 인터페이스)의 기능에 대한 자세한 연습을 제공합니다.
- https://learn.microsoft.com/ko-kr/windows-hardware/test/wpt/wpa-step-by-step-guide

# Windows Performance Analyzer
- 기본 절차 및 자세한 연습은 WPR 빠른 시작 가이드를 참조하세요. WPR UI에 대한 전체 설명서는 WPR 기능을 참조하세요. 주요 시나리오에 대한 논의는 WPA 시나리오를 참조하세요.

# 콘텐츠

|섹션| Description|
|-|-|
|Windows Performance|Toolkit의 새로운 기능|
|이 릴리스에서 사용할 수 있는 새로운 기능에 대해 설명합니다.
|Windows PerformanceRecorder |WPR에 대한 전체 설명서를 제공합니다. MSDN에는 완전한 명령줄 및 확장 가능한 마크업 언어(XML) 참조가 포함됩니다.|
| Xperf 명령줄 참조 | Xperf에 대한 전체 참조 자료를 제공합니다.|
|커널 추적 제어 API 참조| 기존 스크립트 및 프로필과 이전 버전과의 호환성을 위해 지원되는 ETA 이벤트 추적 API의 확장인 커널 추적 제어 API를 다룹니다.|
|Windows Performance Analyzer| WPR 또는 평가 플랫폼에서 만든 기록을 분석할 수 있도록 WPA에 대한 전체 설명서를 제공합니다.|
| Windows 성능 단계별 가이드 | 일반적인 성능 시나리오를 처리하는 랩을 수행하기 위한 지침을 제공합니다|
