# radare2_reverse-engineering_training
- [UNIX-like reverse engineering framework and command-line toolset](https://github.com/radareorg/radare2)

- LinuxOS Install

```
git clone https://github.com/radareorg/radare2
cd sys
./install.sh
```

- macOS Install
  - https://github.com/radareorg/radare2/blob/master/doc/macos.md

```bash
brew install radare2
```
